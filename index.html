Token 追踪

实时记录每次任务的输入/输出 token 数量和成本，帮助您了解 token 使用情况。

💰

预算监控

自动监控预算使用情况，80% 和 100% 时发出警告，确保不超出预算。

📈

趋势分析

分析最近 5 次任务的成本趋势，了解 token 使用模式，及时调整策略。

🛠️

优化建议

根据 token 使用情况自动生成优化建议，帮助您节省 token 消耗。

使用示例

# 记录 token 使用
node index.js record "修复登录bug" 1500 800 "使用limit参数读取文件"

node index.js analyze

将 Token Tracker 集成到您的项目中，开始优化 token 使用。

1

git clone https://github.com/chatgpt-yunju/tokentracker.git

2

安装依赖

cd token-tracker
npm install

3

开始使用

node index.js record "任务名称" 输入token 输出token "优化措施"
